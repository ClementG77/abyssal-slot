# Abyssal Animation Reference Bundle

This bundle is a shareable snapshot of the slot animation reference and the related source/assets it points to.

Start here:

1. Read `files/docs/ABYSSAL_SLOT_ANIMATION_REFERENCE.md`.
2. Open `MANIFEST.md` for the grouped file index.
3. Use the mirrored paths under `files/` to inspect the actual implementation.

The files are copied from the current worktree and keep their original repo-relative paths. For example:

- `files/src/game/bookEventHandlerMap.ts`
- `files/src/components/Symbol.svelte`
- `files/static/assets/wins/tumble_win.png`

This folder is meant for review and handoff, not as a runnable standalone app. The source files still import workspace packages and app-local aliases that only resolve inside the full repository.

Snapshot date: 2026-07-02.
